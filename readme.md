# BPL Medical Tech Scraper

Specialized web scraper for extracting medical technology product data, specifications, and images from industry portals.


## Features
- Structured Data Extraction: Outputs clean JSON files for easy integration.
- Image Downloader: Automatically scans and saves high-res images.
- Data Cleaning: Filters and formats raw HTML into usable insights.


## Tech Stack
- Python / Jupyter
- BeautifulSoup 4 / Selenium
- Pandas
